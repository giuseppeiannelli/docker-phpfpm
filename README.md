# docker-phpfpm

[Docker image](https://hub.docker.com/r/giuseppeiannelli/phpfpm/) for [php-fpm](https://php-fpm.org/) with additional modules:
- exif
- gd
- mcrypt
- mysqli
- mongodb
- pcntl
- pdo_mysql
- rdkafka
- redis
- soap
- sockets
- zip

The image include also [composer](https://getcomposer.org/)
